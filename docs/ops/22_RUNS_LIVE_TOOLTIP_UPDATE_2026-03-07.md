# Runs Live KPI Tooltip Update (2026-03-07)

## What changed
- Added beginner-friendly tooltip glossary on `/runs/live` in the `핵심 지표 / 리스크 모니터` area.
- Added tooltip entries for:
  - `MDD`
  - `리스크 차단`
  - `일시정지`
  - `시퀀스 누락`
  - `킬스위치`
  - `Profit Factor`

## Why
- New users who are unfamiliar with coin trading metrics need immediate in-context explanations.
- This reduces misinterpretation of risk metrics during strategy experiments.

## UI behavior
- Tooltip entry point moved to each metric label (right-side `?`), instead of a separate glossary tag row.
- Existing KPI values and metric calculations remain unchanged.
